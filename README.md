#  RPN Calculator (Калькулятор ОПЗ)

Пользовательский консольный калькулятор для вычисления арифметических выражений, записанных в **обратной польской записи (постфиксной форме)**.

**Вариант:** 15

---

## Основные возможности

* **Интерактивный режим:** Ручной ввод выражений в консоли с мгновенным выводом результата.
* **Пакетный режим:** Считывание пула математических выражений из внешнего файла (`input.txt`).
* **Логирование сессий:** Автоматическая дозапись всех успешных вычислений в файл истории (`history.txt`) в виде готовых равенств.
* **Отказоустойчивость:** Многоуровневая валидация токенов. Перехват критических ошибок (деление на ноль, пустой ввод, некорректная структура ОПЗ) без падения приложения.
* **Чистый ООП:** Собственная изолированная структура данных `Stack` без использования сторонних зависимостей.

---

## Архитектура проекта

Программа разделена на независимые логические модули в соответствии с принципом единственной ответственности (SRP):

* `main.py` — Главная точка входа. Управляет циклом интерактивного консольного меню.
* `calculator.py` — Вычислительное ядро. Содержит класс `RPNCalculator` для парсинга и расчета токенов.
* `stack.py` — Модуль структуры данных. Реализует классический стек по принципу LIFO.
* `file_manager.py` — Файловый менеджер. Изолирует логику безопасного чтения и записи данных.
